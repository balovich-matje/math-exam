# ОГЭ Математика — Telegram Mini App

Тренажёр для подготовки к ОГЭ по математике (9 класс). Теория, практика и тесты по всем заданиям.

## Покрытие заданий

| Задания | Статус |
|---------|--------|
| 1–19 | Готово (Часть 1 — краткий ответ) |
| 20, 21, 23 | WIP — только числовой ответ, на ОГЭ требуется полное решение (Часть 2) |
| 22, 24, 25 | Не реализованы — доказательства и построения, нужен другой формат |

Каждое задание включает:
- **Теория** — правила и разобранные примеры с формулами (KaTeX)
- **Практика** — 3 задачи для тренировки
- **Тест** — 5 задач с подсчётом результата

## Запуск

```bash
npm install
npm run dev
```

Приложение откроется на `http://localhost:5173`.

### Настройка бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. `/newbot` — создайте бота, задайте имя и username
3. Сохраните токен бота
4. `/mybots` → выберите бота → **Bot Settings** → **Menu Button** → укажите URL приложения
5. Или: `/mybots` → **Web App** → **Configure Mini App** → укажите URL

### Локальное тестирование через ngrok

```bash
npx ngrok http 5173
```

Скопируйте HTTPS-ссылку и укажите её в BotFather.

## Сборка

```bash
npm run build
```

Собранные файлы будут в папке `dist/`.
