# ОГЭ Математика — Telegram Mini App

Тренажёр для подготовки к ОГЭ по математике (9 класс). Теория, практика и тесты по всем заданиям.

## Покрытие заданий

| Задания | Статус |
|---------|--------|
| 1–19 | Готово (Часть 1 — краткий ответ) |
| 20, 21, 23 | WIP — только числовой ответ, на ОГЭ требуется полное решение (Часть 2) |
| 22, 24, 25 | Не реализованы — доказательства и построения, нужен другой формат |

Каждое задание включает:
- **Теория** — правила и разобранные примеры с формулами (KaTeX)
- **Практика** — 3 задачи для тренировки
- **Тест** — 5 задач с подсчётом результата

## Текущее состояние проекта

Сейчас в репозитории есть:

- **Frontend Telegram Mini App** на React + TypeScript
- **Онбординг** при первом запуске
- **Главный экран** со списком тем
- Маршруты для **теории**, **практики** и **тестов**
- Раздельный рендер для обычных тем и блочных тем
- **Paywall** для платных тем
- Экран **privacy**
- Локальное хранение части клиентского состояния
- **Backend** на Express
- Серверные маршруты для **auth**, **progress**, **subscription**
- Подключённый **Telegram bot polling** на стороне сервера
- Локальная база данных в папке `server/`
- Собранный frontend в `dist/`

## Структура

- `src/` — клиентское приложение
- `server/` — backend, bot polling и данные
- `public/` — статические файлы
- `pdfs/` — материалы проекта
- `reference/` — вспомогательные файлы и референсы
- `dist/` — собранная версия frontend

## Стек

- Vite + React + TypeScript
- Tailwind CSS
- KaTeX (рендер формул)
- Telegram WebApp SDK (@twa-dev/sdk)
